# **App Name**: NİCAT Portfolio

## Core Features:

- Dynamic Content Display: Showcase Hero section, About Me, Skills, Achievements, and Education, using pre-filled editable placeholder content.
- Project Showcase: Display GitHub projects and descriptions in an organized, card-based layout.
- Certificate Viewer: Present certificates in a card-based format, allowing users to click and view uploaded images or PDFs in a modal.
- Contact Information: Provide readily accessible contact details including email, LinkedIn, and phone number.
- Dark/Light Mode Toggle: Allow users to switch between dark and light themes for personalized viewing experience.
- Content Management System (Placeholder): Basic capability to easily update all text content (e.g., project descriptions, skill details) without code changes.
- AI Project Description Summarizer: A generative AI tool that assists the developer by generating concise project summaries from longer descriptions for optimal display on project cards.

## Style Guidelines:

- A sophisticated dark mode as primary. Background color: Very dark charcoal with a subtle cool undertone (#111516). Primary color: A vibrant yet professional blue for text and main elements (#66BFE6). Accent color: A clear, energetic cyan for interactive elements and highlights (#19E6E6). This palette creates a technical, modern feel, while maintaining high contrast and readability. The primary dark scheme will be designed to easily adapt to a complementary light scheme.
- The 'Inter' typeface, a modern grotesque sans-serif, will be used for both headlines and body text. Its clean, neutral, and objective aesthetic is well-suited for a professional developer's portfolio, ensuring high readability and a contemporary feel across all content.
- Use a consistent set of minimal, outline-style icons. These icons will maintain the clean and professional aesthetic without visual clutter, guiding users intuitively through the portfolio sections.
- Implement a clean, organized, card-based layout for showcasing projects, achievements, and certificates. The design will prioritize responsiveness, ensuring an optimal viewing experience on both mobile devices and large desktop displays through a flexible grid system.
- Subtle and smooth transition animations will be incorporated for interactions such as hovering over cards, opening modals, and navigating between sections, enhancing the user experience with a polished feel without being distracting.
